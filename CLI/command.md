# CLI

command line interface 명령어를 정리합니다. 



#### pwd

현재 디렉토리 위치 출력

```bash
pwd
```



#### ls

list의 약자로 현재 폴더에 있는 모든 파일과 폴더를 출력하는 명령어입니다.

```bash
ls   #숨김폴더 출력 안함
ls -a# 숨김 폴더, 파일까지 모두 출력함
```



#### cd

change directory의 약자로 폴더를 이동하는 명령어입니다.

```bash
cd <PATH> # 이동하고 싶은 폴더위치를 <PATH> 자리에 입력
cd ..     # 상위 디렉토리로 이동
cd ~ 	  # 유저 디렉토리로 이동(home)
cd /	  # root 디렉토리로 이동
```



#### 생성

- ``touch ``: file 생성
- ``mkdir`` : 폴더를 생성
